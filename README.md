# To-Do List App ✅  

## 🚀 Live Demo  
🔗 Website Link (https://todo-app-peach-eta-86.vercel.app/)

## 📌 Features  
- Add, edit, and delete tasks  
- Mark tasks as completed  
- Saves tasks using LocalStorage (data stays after reload)  
- Responsive design with TailwindCSS  

## 🛠️ Tech Stack  
- HTML  
- Tailwind CSS  
- JavaScript  
- LocalStorage
